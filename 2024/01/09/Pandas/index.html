DataFrame与Series

  • Series，仅表现为单独的一列内容 或者 一行内容（图中蓝色部分）
  • DataFrame，不仅可以表现为单独的一列/一行内容，还能表现为多列/行内容，或者是一整张表格

读取与导出数据

从每个数据源导入数据是由前缀为 read_* 的函数提供的。类似地，to_* 方法用于存储(导出)数据。

Pandas提供了read_excel、read_csv等读取数据的方法。当然还有read_json、read_sql，可以自行探索。

data = pd.read_excel(r'1、系统认识数据分析.xlsx')
data.to_excel('test.xlsx')

info可以查看字段类型情况，以此确保你接下来对进行数据处理时，是否需要转换类型

data.info()

在实际读取场景中，DataFrame容易将原本Excel中为文本数值的字段类型变为数值类型，我们可以使用DataFrame的dtype属性来查看各个字段的类型并更改

data = pd.read_excel(r'1、系统认识数据分析.xlsx', converters={'uid': str})

导出时需要保证某些字段的类型如你所期望，使用astype。处理完数据并导出后，我们发现Id字段不应该是数值类型，因此在导出前需要使用astype转换一下类型

data['id'] = data['id'].astype(str)
data.to_excel('test.xlsx',index = False)

筛选数据内容

选择某（些）列

data['id']
data[['id', 'content']]

通过bool索引的方式筛选我们想要的行（比较运算符）。[]会返回判断为True的行目

#选择出为true的行目
data[data['progress'] > 1185000]
data[(data['progress'] > 1185000) & (data['progress'] < 1740000)]

iloc

使用iloc选择行目数 或 具体某个值，相当于limit

data.iloc[0]
# 'select * from data limit 1'
data.iloc[1284:2125]
# 'select * from data limit 1284, 2125'

loc依据索引标签进行筛选 iloc依据索引位置进行筛选。用 loc[] 或者 [] 来调取数据内容，如果没有重置索引的话，将会出现严重的问题，而 iloc[] 不会受到 未重置索引 的影响

data_concat.loc[100]
# data_concat是后面纵向合并、没重设索引标签的结果
# 会有两个100的行
data_concat.iloc[100]
# 第100行只有一个

for循环

for i in range(len(data)):
    print(data['id'].iloc[i])

for danmu_id in data['id']:
    print(danmu_id)

数据去重

调用drop_duplicates()方法，并将需要去重的列作为参数传入

data.drop_duplicates(subset='uid',keep = 'first',inplace=True)
# 使用uid列的unique值 同值保存第一个，或设置'last'等，inplace设置是否赋值、改变原变量

数据合并、连接

Pandas用类似于SQL的连接、合并操作，可以横向、纵向拼接多份数据

轴（axis）

  • 当我们参数指定为axis=0的时候，效果作用于行，即行数会发生变化 或 效果自上而下
  • 当我们参数指定为axis=1的时候，效果作用于列，即列数会发生变化 或 效果自左往右
df = pd.DataFrame([{'a':1, 'b':2, 'c':3},
                  {'a':4, 'b':5, 'c':6},
                  {'a':7, 'b':8, 'c':9}])
print(df)
a b c
0 1 2 3
1 4 5 6
2 7 8 9 
df.sum(axis=0)
#a    12
#b    15
#c    18
#dtype: int64
df.sum(axis=1)
#0     6
#1    15
#2    24
#dtype: int64

拼接

  • 横向拼接，列数发生变化
pd.concat([pd1, pd2], axis=1)
  • 纵向拼接，行数发生变化
pd.concat([pd1, pd2], axis=0)

一般纵向拼接之后，索引都会出现问题，需要采用reset_index()对索引进行重置

data_concat.reset_index()

表连接

利用pd.merge()实现表连接

inner = pd.merge(left=pd1, right=pd2, how='inner', left_on='uid', right_on='uid')

DataFrame增加列的方式与字典（dict）新增键值对的方式相同

data.rename(columns={'id': '弹幕id', 'content': '弹幕'}, inplace=True)

排序与聚合分组

函数 作用 参数 备注
sort_values 排序 by：用以指定参与排序的列，ascending默认为True ascending：选择降序或正序 使用范例1：data.sort_values('字段名', ascending=False) 使用范例2：data.sort_values(['字段名1', '字段名2'])
groupby 聚合分组 by：用以指定参与排序的列 需要再指定哪种聚合函数
agg 指定多个不同的聚合函数 groupby('维度').agg(['count', 'sum'])：使用方式示例1 groupby('维度').agg({'度量1':'sum', '度量2': 'count'})：使用方式示例2 groupby('维度').agg({'度量1':['count', 'sum'], '度量2': 'count'})：使用方式示例3
transform 能达到SQL窗口函数作用的聚合计算 使用方式示例：groupby('维度')['度量'].transform('sum') 
data.sort_values(by='uid')
data.sort_values(by=['uid', 'ctime'], ascending=False)
data.sort_values(by=['uid', 'ctime'], ascending=[True,False])

rank

rank函数中的methods参数

  • max：最大值排名，即相同值的排名都是最大的那个值。
  • min：最小值排名，即相同值的排名都是最小的那个值。
  • average：平均排名，即相同值的排名是所有相同值排名的平均值。
  • dense：密集排名，即相同值的排名是连续的，没有空缺。
  • first：首次出现排名，即相同值的排名是它们在数据中首次出现的位置。

average

data = pd.DataFrame({'dept': ['A', 'A', 'B', 'B', 'B'], 'salary': [100, 200, 300, 300, 400]})
data['row_number'] = data.groupby('dept')['salary'].rank(ascending=False, method='average')

map、apply、applymap

在padnas中，Series类型才能使用map函数。使用map函数时，传入参数可以是函数 或者是 字典

【map操作的对象是某列】

# 传入字典
df['中文名'] = df['Animal'].map({'战鹰': 'eagle', '捷豹': 'cat'})
# 【注意】不在字典里的会是NaN
# 传入匿名函数
df['【匿名】Max Speed'] = df['Max Speed'].map(lambda x: x+5) #
# 传入函数
df['【函数】Max Speed'] = df['Max Speed'].map(add)

在pandas中，Series、DataFrame类型都可以使用apply函数。当你需要对多个不同的列，进行不同的操作时，可以使用

def func(series):
    if series.name == 'Animal':
        parse = series.map({'战鹰': 'eagle', '捷豹': 'cat'})
        return parse
    elif (series.name == 'Max Speed') or (series.name == 'Min Speed'):
        parse = series + 10
        return parse
# 将animal字段转换为英文；将所有数值字段都 + 10
df.apply(func)

apply函数传参：axis 参数默认为0/列。可以指定axis=1

  • 0 或者 index ，表示函数处理的是【每一列】；
  • 1 或 columns ，表示处理的是【每一行】;

在pandas中，DataFrame类型才能使用applymap。

【注意】在使用applymap()函数时，需要确保函数是逐元素操作的，否则可能会导致错误。

# 将所有的值都转换为字符串类型
def to_str(values):
    return str(values)
df = df.applymap(to_str)
